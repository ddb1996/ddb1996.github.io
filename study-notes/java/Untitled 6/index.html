如果不声明，默认小数为 double 类型，所以如果要用 float 的话，必须进行强转

例如：float a=1.3; 会编译报错，正确的写法 float a = (float) 1.3; 或者 float a = 1.3f;（f 或 F 都可以不区分大小写）

java 方法：

访问修饰符 + 返回类型 + 方法名（参数，int... 可变参数列表）{

语法体

}

void 表示没有返回数值

由此可分为四种方法类型，无参有值，无参无值 有参有值，有参无值。

数组也可以作为方法当中的参数

可变参数列表可以有数组传递，但是反过来不可以

方法可以重载，重载的的定义是方法名相同、参数列表不同（参数个数、参数类型）

java -d doc XXX.java 编译相应的 java 程序文档注释

数组作为方法参数传值调用后会改变原来主方法当中定义数组的值。

因为数组属于引用数据类型、数组名指向了数组当中第一个元素、

方法当中的数组名也指向了数组当中的第一个元素，所以两者是等价的，指向同一个内存地址，只是名称不同。所以调用后，数组中元素的值会被改变。

基本数据类型 常量 被作为方法当作参数传递的时候，只是把主方法当中的值传递给了方法中的变量。所以调用后，主方法的常量值不会改变

关于调试，F 6 单部调试，F5 进入到调用方法中调试，F7 返回到调用前方法

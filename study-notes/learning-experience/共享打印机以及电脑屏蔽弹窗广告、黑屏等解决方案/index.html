# 一．电脑黑屏相关问题

电脑连线位置图：

1. 电脑开机黑屏问题（键盘灯亮）可通过三种方式排除解决:
①配有独立显卡的主机 应将 VGA 线（图 1）接到机箱后部下方独立显卡处连接 即可解决开机黑屏问题。

②没有独立显卡的主机多半是配置较差、老旧的机型。如遇开机黑屏问题，可用螺丝刀拆开机箱侧盖拔掉内存条（图 2）用橡皮擦拭金手指在插回主机即可。（如果还不能解决只能申请更换电脑主机）③拔掉与主机相连接的 U 盘，长按电源键 5 秒强制关机
重新开机即可。

# 二、电脑开机慢问题

1. 可使用 360 等安全杀毒软件禁止开机启动项，清理垃圾、杀毒后得到一定程度上的缓解。

建议卸载不需要的软件，软件尽量不要都安装在 C 盘，安装时注意安装附带选项，不要勾选开机启动、安装其他附带软件

如果以上步骤都进行过了还是不能解决问题，建议备份好电脑上重要的数据至 U 盘或除 C 盘以外的盘后，使用自带联想一键恢复软件恢复系统。（开机后持续敲击键盘 F2 键）或使用其他方式安装系统。

# 四．打印机共享问题。

共享打印机能够长期正常使用的前提：

1. 确保共享打印机的主机开机、ip 地址不变。

2. 确保打印机与主机的连线正常 **、打印机开机。**

3. 确保打印机驱动正常。一般打印机连接后等待少许，驱动会自动安装完毕。现阶段局内有两台打印机驱动除外（HP P1106 和 OKI５８６０SP　需百度下载相关驱动自行安装驱动。）

4. 共享打印机主机能够正常打印后再去设置连接打印机的电脑。

共享打印机主机需进行的设置：

１. 按 WIN（有微软图标的键）＋R，在输入框里输入 CMD

在 CMD 窗口下输入ｉｐｃｏｎｆｉｇ命令查看自己当前 IP 地址，告诉其他连接打印机的人你的 IP 地址。

２. 剩余操作如下图所示：

最后点击保存修改

３. 按 WIN 键在开始菜单栏找到设备和打印机，找到自己的打印机右键设置为默认打印机，再次右键选择打印机属性。

在弹出的窗口中标题栏中点击共享，选择共享此台打印机。

设置完成后点击打印测试页看看是否打印正常。

连接打印机主机需进行的设置：

１．按 WIN（有微软图标的键）＋R，在输入框里输入共享打印机主机的 IP 地址。格式如下：＼＼１９２．１６８．＊．＊

２．在弹出的窗口中找到相应的打印机双击连接即可。然后测试打印机能否正常使用。

=**=** 以下内容没有出现输入密码提示不需要看 ==

少数连接打印机的电脑会出现如下情况（以下操作是为了解决重启后连接打印机的电脑不能打印问题 **，可能是来宾账户、控制台登录导致的 **）：

在弹出的窗口中出现输入账号密码的提示，而共享打印机的主机又没有设置密码。输入账户 lenovo 后，密码不填，勾选记住选项即可。（lenovo 若不行试试 Administrator 或者 Guest（低权限账户））

然后点击 WIN 键找到控制面板 ****。在下图搜索框中输入凭据管理器。在 windows 凭据下会自动生成一个凭据。如果没有生成 windows 凭据 则需要点击添加 windows 凭据手动添加。

# 五．打印机故障类

出现如图所示故障时：重启打印机主机和连接打印机的主机即可。(可能是防火墙关闭导致。)

# 六．打印机任务无法取消等问题通用方法

若打印机有停止、清除等按键请先按下停止、清除键。

（没有进行下一步）

按 WIN＋R，在输入框里输入以下命令：services.msc

在弹出的窗口中按 P 键找到 Print Spooler 服务，右键单击停止服务。

按 WIN＋R，在输入框里输入以下以下路径：

C:\Windows\System32\spool\PRINTERS

删除该路径下所有文件（若没有文件跳过进行下一步）

切换到服务界面，右键单击启动 “printspooler” 服务；
